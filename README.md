# Coin Toss App

A simple and interactive coin toss application built with Expo and React Native.

## Demo

https://github.com/user-attachments/assets/prototype.mp4

Or view the demo video: [prototype/prototype.mp4](./prototype/prototype.mp4)

## Features

- Random coin toss with heads and tails animations
- Video playback for each toss result
- Clean and intuitive user interface
- Dark/Light mode support

## Installation

```bash
npm install
```

## Run the app

```bash
npm start
```

Then scan the QR code with Expo Go app on your mobile device.
